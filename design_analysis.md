# Culture Gem Website Design Analysis

## Key Design Elements

### Color Palette
- **Primary Green**: Dark forest green (#2D6B4F or similar) - used for buttons and CTAs
- **Accent Colors**: 
  - Soft blue/slate blue for highlighted text
  - Warm peachy/orange for accent highlights
  - Soft pink/mauve for additional highlights
- **Background**: Clean white/off-white
- **Text**: Dark charcoal/black for body text

### Typography
- **Headlines**: Bold, sans-serif (appears to be a modern sans-serif)
- **Body Text**: Clean, readable sans-serif
- **Hierarchy**: Strong visual hierarchy with large, bold headlines

### Layout & Structure
- **Navigation**: Horizontal top navigation bar with logo on left
- **Hero Section**: Split layout with illustration on left, text content on right
- **Illustration Style**: Playful, line-drawn characters and scenes (not photographic)
- **Spacing**: Generous whitespace, clean and uncluttered
- **Content Sections**: Clear separation between sections

### Visual Style
- **Illustrations**: Hand-drawn, line-art style with simple shapes
- **Color Blocks**: Colored backgrounds/highlights behind key text phrases
- **Buttons**: Solid color with subtle shadows, rounded corners
- **Overall Feel**: Modern, professional yet approachable; playful but trustworthy

### Navigation Structure
- Logo/brand on top left
- Horizontal menu items across top
- CTA buttons (Get In Touch, Book a demo) on right side
- Clean, minimal design

### Key Observations
1. **Asymmetric Layout**: Hero uses left-right split rather than centered layout
2. **Colored Text Highlights**: Key phrases are highlighted with soft background colors
3. **Playful Illustrations**: Custom line-art illustrations add personality
4. **Generous Whitespace**: Lots of breathing room between elements
5. **Clear Hierarchy**: Large, bold headlines guide attention
6. **Professional Green**: The primary green conveys trust and professionalism

## Design Philosophy for grctoisms
- Clean, modern consulting aesthetic
- Playful but professional tone
- Strong use of the forest green color
- Generous spacing and whitespace
- Asymmetric, dynamic layouts
- Custom illustrations or simple graphics
- Clear navigation and CTAs
